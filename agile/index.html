Code conventions serve the following purpose:

  • A consistent look to the code
  • Code that is easier to read and follow
  • Facilitates the maintenance of code

Naming

  • Choose easily readable names, use good English
  • Favour readability over brevity
  • Do not use underscores, hyphens, or other non alphanumeric characters
  • Do not use Hungarian notation
  • Avoid using identifiers that conflict with language keywords
  • Avoid using abbreviations or contractions
  • Avoid using acronyms that are not widely understood and accepted
