Naming

The following should be upper camel case (UpperCamelCase):

  • Types and namespaces
  • Methods, properties and events
  • Constants
  • Enum members
  • Fields, when not private

The following should be lower camel case (lowerCamelCase):

  • Local variables
  • Local constants
  • Parameters

The following should be underscore lower camel case (_lowerCamelCase):

  • Fields, when private

Interfaces should be upper camel case prefix with I (IUpperCamelCase).

Generic types parameters should be upper camel case prefix with T (TUpperCamelCase).

{% highlight c# %} namespace CreativeJar.Standards { public interface INaming {} public class Naming : INaming { private string _myPrivateField; public string MyPublicField; public Naming() { //... } public string MyProperty { get; set; } public TName MyMethod() { return default(TName); } public enum MyEnum { ItemOne, ItemTwo, ItemThree } } } {% endhighlight %}
